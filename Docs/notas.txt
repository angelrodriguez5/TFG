### Dejar procesos corriendo en la máquina:

- Usar el comando para abrir una consola cuyos procesos
no terminan al cerrarla

	screen

ejecutar procesos en la consola y cerrar usando la 'X'

para visualizar esa consola de nuevo, volver a conectar al
equipo y usar:

	screen -r [proceso]

para auto-completar el proceso usar el tabulador
